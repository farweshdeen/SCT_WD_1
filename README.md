Task 01: Responsive Landing Page

Project Description

This project was developed as part of Task 01 of the SkillCraft Technology Web Development Internship.

The objective of this task was to design and develop a responsive landing page with an interactive navigation menu using HTML, CSS, and JavaScript. For demonstration purposes, the landing page is themed around VAAN Foundation, an astronomy outreach organization, initiated by me for showcasing a demonstrate public astronomy programme announcement.

Features
Responsive design for desktop and mobile devices
Fixed navigation menu
Navigation bar changes appearance on scroll
Interactive hover effects
Clean and modern user interface
Smooth user experience

Technologies Used
1.HTML5
2.CSS3
3.JavaScript
